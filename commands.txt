---------------SET ENCODER---------------------------------------------------------
E <profile> <n encoder> <type> <value> <value optionel> <value optionel>
command arg0   arg1      arg2    arg3        arg4              arg5

type:
 - 0 : System Action
 - 1 : Key Combination UP
 - 2 : Key Combination DOWN
 - 3 : Key Combination CLICK
 - 4 : Midi

E 0 0 0 0 #Master volume
E 2 0 1 #Next/previos

E n 2 65 #KeyA
E n 2 65,66 #KeyA + KeyB

-----------------------------------------------------------------------------------



----------------SET KEY------------------------------------------------------------
K <profile> <n key> <type> <value> <value optionel> <value optionel>
command  arg0    arg1   arg2        arg3             arg4
type:
 - 0 : System Action
 - 1 : Key Combination
 - 2 : Midi

K 0 1 182 #previous
K 1 1 205 #PlayPause
K 2 1 181 #Next

K 3 1 179
K 4 1 183
K 5 1 179

K 0 1 65 #KeyA
K n 2 65,66 #KeyA + KeyB
